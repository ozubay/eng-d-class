# Graphic Design English Trainer

## 현재 상태
- `index.html` + `assets/` : Manus 배포본(프로덕션 빌드)을 그대로 호스팅 (커스텀 도메인 루트 서빙).
- `404.html` : SPA 딥링크 fallback (index.html 복사본).
