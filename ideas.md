## Idea A: Neo-Minimal Workspace (확률: 0.07)

**Design Movement**: Neo-Minimal SaaS Dashboard — HubSpot/Linear 계열의 전문 업무 도구 미학

**Core Principles**:
1. 오프화이트 배경 위에 순백 카드가 부유하는 레이어드 뉴트럴 팔레트
2. 라임 그린 단일 액센트로 진행 상태와 성취를 시각화
3. 다크 사이드바와 밝은 콘텐츠 영역의 명확한 분리
4. 기능이 곧 장식 — 모든 시각 요소는 학습 상태를 전달하는 역할을 가짐

**Color Philosophy**: 거의 흰색에 가까운 배경(#F5F5F4)에 순백 카드, 그리고 라임 그린(#A8E63D) 하나만 액센트로 사용. 이 색은 "살아있음", "진행 중", "달성"을 의미하며 게임화 요소와 전문성을 동시에 전달.

**Layout Paradigm**: 좌측 64px 다크 사이드바 + 우측 오프화이트 콘텐츠 영역. 상단에 다크 필 형태의 진행 바. 카드 그리드는 비대칭적으로 배치되어 시각적 흐름을 만듦.

**Signature Elements**:
1. 라임 그린 진행 필 (상단 타임라인 바)
2. 다크 원형 아이콘 버튼 사이드바
3. 활성 미션 카드의 라임 그린 전체 채움

**Interaction Philosophy**: 완료 시 카드가 라임으로 변하고, 오답 시 빨간 쉐이크, 정답 시 초록 플래시. 진행 상태가 항상 상단 바에 표시.

**Animation**: 카드 호버 180ms lift, 정답 300ms 플래시, 오답 400ms 쉐이크, 스테이지 언락 500ms 팝.

**Typography System**: DM Sans (헤딩, 800/700/600) + Noto Sans KR (본문, 400/500) + JetBrains Mono (키워드 하이라이트)

## Idea B: Editorial Brutalism (확률: 0.06)

**Design Movement**: Editorial Brutalism — 타이포그래피 중심의 강렬한 흑백 베이스에 형광 노랑 액센트

**Core Principles**:
1. 타이포그래피가 레이아웃의 주요 구조 요소
2. 두꺼운 보더와 날카로운 직각 코너
3. 고대비 흑백 + 형광 1색 액센트
4. 그리드를 의도적으로 깨는 오버랩 요소

**Color Philosophy**: 순백 배경, 순흑 텍스트/보더, 형광 노랑(#FFE600) 액센트. 디자인 교육의 바우하우스 정신을 반영.

**Layout Paradigm**: 상단 고정 헤더 + 타이포그래피 기반 섹션 구분. 카드 없이 보더로만 구역 구분.

**Signature Elements**:
1. 두꺼운 흑색 보더 (4px)
2. 형광 노랑 하이라이트 텍스트
3. 대형 숫자 타이포그래피로 진행 상태 표시

**Interaction Philosophy**: 클릭 시 색상 반전, 완료 시 형광 노랑 채움.

**Animation**: 즉각적인 색상 반전, 최소한의 모션.

**Typography System**: Space Grotesk (헤딩, 800) + IBM Plex Sans (본문) + IBM Plex Mono (코드)

## Idea C: Glassmorphism Dark Academy (확률>0.04)

**Design Movement**: Dark Academy Glassmorphism — 딥 네이비/차콜 배경에 반투명 유리 카드

**Core Principles**:
1. 어두운 배경 위 반투명 카드의 깊이감
2. 그라데이션 액센트 (청록→라임)
3. 학문적 권위와 게임화의 결합
4. 별자리/지도 은유의 스테이지 맵

**Color Philosophy**: 딥 네이비(#0F172A) 배경, 반투명 카드(white/10%), 청록→라임 그라데이션 액센트.

**Layout Paradigm**: 중앙 집중형 콘텐츠 + 플로팅 카드 그리드. 별자리 형태의 스테이지 맵.

**Signature Elements**:
1. 반투명 유리 카드 (backdrop-blur)
2. 청록→라임 그라데이션 진행 바
3. 별자리 연결선으로 스테이지 관계 표시

**Interaction Philosophy**: 호버 시 카드 글로우 효과, 완료 시 파티클 애니메이션.

**Animation**: 글로우 펄스, 파티클 폭발, 부드러운 블러 전환.

**Typography System**: Sora (헤딩) + Noto Sans KR (본문)
